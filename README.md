# hexo-lunar-calendar [![npm](https://img.shields.io/npm/v/npm.svg)](https://nodejs.org/en/)
Lunar calendar for [hexo](https://hexo.io).

## Install
```
$ npm i hexo-lunar-calendar
```

## Usage
```
<%= dateLunar(date(post.date)) %>
```
